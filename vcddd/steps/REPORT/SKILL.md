# Step: REPORT — 最终报告

汇总所有域的完成情况和集成验证结果，生成供用户阅读的全局快照。

## 前置条件

- 全部域代码实现已完成并通过审查
- 跨域集成验证已完成

## 输入

- 各域的 `progress.log`、集成验证报告

## 输出

向用户呈现的完成摘要

## 执行流程

1. 汇总各域状态（DONE / DONE_WITH_CONCERNS）
2. 汇总各域测试结果
3. 汇总各域审查记录和遗留问题
4. 汇总集成验证结果
5. 列出所有遗留问题

## 如果卡住

读 `../../reference/engine/automated-execution.md` > "最终报告格式"。

## 验证

- [ ] 各域状态与审查记录一致
- [ ] 测试数量与实际文件一致
- [ ] 遗留问题完整列出
- [ ] 代码位置标注了各域的目录
