# VCDDD — For Human

本目录保存面向人类阅读的 VCDDD 理论、哲学背景和方法演进材料，不属于 AI Skill 的运行时上下文。

## 建议阅读顺序

1. `vcddd-whitepaper.md`：理解 VCDDD 为什么出现，以及它试图保护什么。
2. `vcddd-methodology.md`：理解 VCDDD 的核心命题与完整方法框架。
3. `vcddd-design-guide.md`：理解业务事实、决策主权、边界与协作契约。
4. `vcddd-implementation.md`：理解理论如何曾经被映射到工程实践。

## 与 AI Skill 的边界

- `For Human/` 负责解释思想、历史和方法演进，允许保留较长论述。
- 2.0 init 暂不包含 AI Skill，当前只建立面向人类的理论基线。
- 未来的 `vcddd/SKILL.md` 只负责教 AI 在当前任务中如何思考和行动。
- AI Skill 不依赖本目录，不应为了执行普通任务自动加载这些文件。
- 本目录中的旧流程或工程策略不自动构成当前 Skill 的强制规则。

当前四份理论文件先按原版本保存。后续可以继续标注哪些是长期哲学，哪些属于已经废弃的流程设计。
