# VCDDD 1.0 — For Human

本目录保存面向人类阅读的 VCDDD 1.0 理论、哲学背景和方法材料。它们是 VCDDD 2.0 的历史思想资源和反思对象，不自动构成 2.0 的结论或约束。

## 建议阅读顺序

1. `vcddd-whitepaper.md`：理解 VCDDD 为什么出现，以及它试图保护什么。
2. `vcddd-methodology.md`：理解 VCDDD 的核心命题与完整方法框架。
3. `vcddd-design-guide.md`：理解业务事实、决策主权、边界与协作契约。
4. `vcddd-implementation.md`：理解理论如何曾经被映射到工程实践。

## 材料边界

- 四份理论文件按 1.0 原版本保存，用于理解当时的理论主张与工程映射。
- 其中的固定概念、流程门禁和实现策略不应被默认继承到 2.0。
- 2.0 的新思考与阶段性结论保存在 `doc/2.0/`。
